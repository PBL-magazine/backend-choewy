# Summary

- Method: GET
- Url: /api/users/auth

# Request

```json
{
  "headers": {
    "Authorization": "Bearer ${ token }"
  }
}
```

# Response

## Success

```json
{
  "status": "200 Ok",
  "data": {
    "ok": true,
    "user": {
      "user_id": "String",
      "email": "String",
      "nickname": "String",
      "role": "Number"
    }
  }
}
```

## Errors

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "로그인 필요"
  }
}
```

```json
{
  "status": "404 Not Found",
  "data": {
    "ok": false,
    "message": "사용자 정보가 존재하지 않음"
  }
}
```

```json
{
  "status": "500 Internal Server Error"
}
```
