# Summary

- Method: POST
- Url: /api/users/signup

# Request

```json
{
  "body": {
    "email": "String",
    "nickname": "String",
    "password": "String"
  }
}
```

# Response

## Success

```json
{
  "status": "201 Created",
  "cookies": { "token": "String" },
  "data": { "ok": true }
}
```

## Errors

```json
{
  "status": "400 Bad Request",
  "data": {
    "ok": false,
    "message": "입력 형식 확인 요망"
  }
}
```

```json
{
  "status": "409 Conflict",
  "data": {
    "ok": false,
    "message": "사용자가 이미 존재함"
  }
}
```

```json
{
  "status": "500 Internal Server Error"
}
```
