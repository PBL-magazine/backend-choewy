# Summary

- Method: POST
- Url: /api/users/signup

# Query

## Find User By Email and Nickname

```sql
SELECT * FROM Users AS User
  WHERE (User.email = '${ email }' AND User.nickname = '${ nickname }')
LIMIT 1;
```

## Create new User

```sql
NSERT INTO Users
  (user_id, email, nickname, password, createdAt, updatedAt)
VALUES (?, ?, ?, ?, ?, ?);
```

# Request

```json
{
  "body": {
    "email": "String",
    "nickname": "String",
    "password": "String"
  }
}
```
