# Summary

- Method: POST
- Url: /api/users/signup

# Request

```json
{
  "body": {
    "email": "String",
    "nickname": "String",
    "password": "String"
  }
}
```

# Response

## Success

```json
{
  "status": "201 Created",
  "headers": {
    "cookies": { "token": "String" }
  },
  "data": { "ok": true }
}
```

## Errors

```json
{
  "status": "400 Bad Request",
  "data": {
    "ok": false,
    "message": "이메일 형식을 확인하세요."
  }
}
```
