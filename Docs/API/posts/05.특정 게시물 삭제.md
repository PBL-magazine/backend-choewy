# Summary

- Method: DELETE
- Url: /api/posts/:post_id

# Request

```json
{
  "headers": {
    "authorization": "Bearer ${ token }",
    "content-type": "multipart/form-data"
  },
  "body": {
    "content": "String",
    "image": "FileData"
  }
}
```

# Response

## Success

```json
{
  "status": "200 Ok",
  "data": { "ok": true }
}
```

## Errors

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "로그인 상태가 아닙니다."
  }
}
```

```json
{
  "status": "404 Not Found",
  "data": {
    "ok": false,
    "message": "해당 게시물 정보를 찾을 수 없습니다."
  }
}
```

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "삭제 권한이 없습니다."
  }
}
```
