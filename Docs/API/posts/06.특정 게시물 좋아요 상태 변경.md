# Summary

- Method: PUT
- Url: /api/posts/:post_id/like

# Request

```json
{
  "headers": {
    "Authorization": "Bearer ${ token }"
  }
}
```

# Response

## Success

```json
{
  "status": "200 Ok",
  "data": { "ok": true }
}
```

## Errors

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "로그인 필요"
}
```

```json
{
  "status": "404 Not Found",
  "data": {
    "ok": false,
    "message": "게시물 정보를 찾을 수 없음"
  }
}
```

```json
{
  "status": "500 Internal Server Error"
}
```
