# Summary

- Method: POST
- Url: /api/posts

# Request

```json
{
  "headers": {
    "Authorization": "Bearer ${ token }",
    "content-type": "multipart/form-data"
  },
  "body": {
    "content": "String",
    "image": "FileData"
  }
}
```

# Response

## Success

### Status Code

```
201 Created
```

### Data

```json
{
  "ok": true
}
```

## Errors

### Status Code

```
400 Bad Request
401 Unauthorized
```

### Data

```json
{
  "ok": false,
  "message": "string"
}
```
