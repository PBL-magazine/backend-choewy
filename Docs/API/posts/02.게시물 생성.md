# Summary

- Method: POST
- Url: /api/posts

# Request

```json
{
  "headers": {
    "authorization": "Bearer ${ token }",
    "content-type": "multipart/form-data"
  },
  "body": {
    "content": "String",
    "image": "FileData"
  }
}
```

# Response

## Success

```json
{
  "status": "201 Created",
  "data": { "ok": true }
}
```

## Errors

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "로그인 상태가 아닙니다."
  }
}
```

```json
{
  "status": "400 Bad Request",
  "data": {
    "ok": false,
    "message": "내용을 입력하세요."
  }
}
```

```json
{
  "status": "400 Bad Request",
  "data": {
    "ok": false,
    "message": "이미지를 입력하세요."
  }
}
```
