# Summary

- Method: POST
- Url: /api/posts

# Query

```sql
INSERT INTO posts(user_id, content, image) VALUES(?, ?, ?);
```

# Request

```json
{
  "headers": {
    "authorization": "Bearer Token",
    "content-type": "multipart/form-data"
  },
  "body": {
    "content": "String",
    "image": "FileData"
  }
}
```

# Response

## Success

```json
{
  "status": "201 Created",
  "data": {
    "ok": true,
    "row": {
      "post_id": "Number",
      "content": "String",
      "image": "String",
      "createdAt": "Date",
      "updatedAt": "Date",
      "author": {
        "user_id": "Number",
        "email": "String",
        "nickname": "String"
      }
    }
  }
}
```

## Errors

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "로그인 상태가 아닙니다."
  }
}
```
