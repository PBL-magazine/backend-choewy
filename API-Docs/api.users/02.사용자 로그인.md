# Summary

- Method: POST
- Url: /api/users/signin

# Request

```json
{
  "body": {
    "email": "String",
    "password": "String"
  }
}
```

# Response

## Success

```json
{
  "status": "200 Ok",
  "headers": {
    "cookies": { "token": "String" }
  },
  "data": { "ok": true }
}
```

## Errors

```json
{
  "status": "400 Bad Request",
  "data": {
    "ok": false,
    "message": "이메일 형식을 확인하세요."
  }
}
```

```json
{
  "status": "400 Bad Request",
  "data": {
    "ok": false,
    "message": "비밀번호 형식을 확인하세요."
  }
}
```

```json
{
  "status": "404 Not Found",
  "data": {
    "ok": false,
    "message": "존재하지 않는 이메일 계정입니다."
  }
}
```

```json
{
  "status": "401 Unauthorized",
  "data": {
    "ok": false,
    "message": "이메일 또는 비밀번호를 확인하세요."
  }
}
```

}
